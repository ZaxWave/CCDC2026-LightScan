# 📁 Datasets Directory

此目录用于存放模型训练数据集。

本项目数据集涵盖了从底层时空数据采集、路面损伤检测到高层 RDD 算子处理的全流程。

## 📊 暂定使用数据集列表 (Road Damage & Objects)

项目初步选定以下公开数据集用于模型训练与验证：

| 数据集名称 | 核心用途 | 特点说明 |
| :--- | :--- | :--- |
| **CNRDD** | 国内标准对齐 | 符合中国《公路技术状况评定标准》(JTG 5210)，包含裂缝、坑洞等国内常见病害。 |
| **RDD2022 / 2024** | 通用特征预训练 | 多国路损挑战赛数据集，样本量大，涵盖全球不同气候和铺装条件下的路损图像。 |
| **GAPs** | 高精度模型验证 | 德国专业级道路影像，分辨率极高，用于对精细裂缝识别能力的基准测试。 |
| **Crack500** | 裂缝专项强化 | 专注于路面裂缝提取，适合用于语义分割（Segmentation）或细粒度分类。 |


> **⚠️ 注意**：为了保护存储空间和避免 Git 仓库过大，此目录下的所有内容（除本说明文件外）均已通过 `.gitignore` 排除在版本控制之外。
