# 📚 LightScan 项目文档库

本目录存放 LightScan 项目全生命周期的所有非代码类文档，包括技术方案、需求分析、调研记录及比赛合规文件。

## 📂 目录导航

### 1. [Research (调研分析)](./research/)
- **算法选型**: 轻量化检测模型技术路线对比分析（LS-Det 架构选型依据）。
- **竞品分析**: 国内外现有路损检测系统的功能对标。

### 2. [Technical (技术积淀)](./technical/)
- **架构设计**: 前后端分离架构图及数据流向说明。

### 3. [Requirements (需求/作品要求)](./requirements/)
- **CCDC 2026 官方规范**: 整理后的比赛核心要求。
- **功能清单 (Backlog)**: 已实现、开发中及待开发的功能模块。

### 4. [Notices & Resources (通知与资料)](./notices/)
- **官方通知**: 比赛关键节点的时间线。
- **参考资料**: 相关的学术论文、技术博客及开源项目引用。

## ✍️ 编写规范
1. **Markdown 优先**: 所有文档建议使用 Markdown 格式编写，以便在 GitHub 上直接阅读。
2. **图表集成**: 复杂的逻辑流程建议使用 Mermaid.js 绘制或存放于根目录 `assets/` 后在此引用。
3. **版本控制**: 重要文档的变更应在 Commit Message 中明确记录（如 `docs: update system architecture diagram`）。
